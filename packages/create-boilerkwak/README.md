# Boilerkwak

Crée une application React, TypeScript, Vite, Tailwind CSS et TanStack Router
à partir de Boilerkwak.

```bash
npm create boilerkwak@latest mon-projet
```

Pour ne pas installer immédiatement les dépendances :

```bash
npm create boilerkwak@latest mon-projet -- --no-install
```

Le template utilisé est
[MaximeSeignovert/boilerkwak](https://github.com/MaximeSeignovert/boilerkwak).

## Publication

```bash
npm login
npm publish
```
